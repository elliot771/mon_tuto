1. La base


1.1 La bonne application pour débutant à installer


Le bon site pour débutant à installer est sublime texte 2. Il est gratuit et il est super pratique. Il se peut que des fois sublime texte vous demande une petite contribution mais ne vous inquiétez pas vous pouvez très bien refuser.

1.2 Le code que nous allons utiliser


Nous allons utiliser deux types de codes. Au début il y aura du HTML, puis ensuite nous utiliserons du Css


1.3 L'utilité du HTML


Le html lui sert a faire le gros de la page comme écrire le texte de la page les titres, les espaces, souligner, surligner, etc.

1.4 L'utilité du Css


Le Css sert a faire en quelque sorte la déco de la page comme mettre un fond en couleur, changer la police des textes, souligner, etc.

2 Commencer à coder du HTML


2.1 Comment faire des pargraphes structuré


Pour commencer à faire du html vous devez enregistrer la page en lui donnant un nom de votre choix puis vous devez impérativement terminé par .html sinon votre ordinateur ne va pas comprendre que vous codez du HTML il faudra faire de même pour css. Ensuite pour faire des paragraphes bien structuré il faut mettre des <> avec un p à l'interieur. Puis écrivez votre texte et finisez par un "" avec à l'interieur un /p pour monter que vous avez finis cette partie du texte. Pour voir votre progression aller sur google et appuyer sur cdm o et cliquer sur votre fichier html.

2.2 Comment mettre des titres


Pour mettre des titres il faut tout simplement mettre des balises (>) avec h1 dedans et finir avec des balises avec /h1. Il y a différentes tailles de titre il y a les h1 (les plus gros), les h2 qui sont un peu plus petit que les h1, les h3 qui sont plus petits que les h2, etc.

2.3 Comment mettre des espaces


pour mettre des espaces rien de plus simple que de mettre 1 balise(>) avec br. Avec la balise br nous n'avons pas besoin de terminer par une balise /br.

2.4 Crée des liens


Crée des liens est un peut plus compliquer, il faut mettre des balises avec à l'intérieur a bref="votre nom de fichier">ici vous écrivez ce qu'il affiche en bleu"/a> et voilà vous avez appris la base du HTML mais seulement la base car pour avancer il vous faut écrire la même chose que sur la photo qui est affiché juste en dessous, si c'est trop petit cliquer dessus. Pour mettre tout ce qui est du texte les titre(h1, h2, ...) ce sera dans body. Et la balise head sert à mettre le titre de la page, etc.

2.5 Insérez des images


Pour que vous insériez des images il faut que vous marquiez dans des balises "img Src=" le dossier/le nom de l'image" alt="la description de l'image pour les robots et les aveugles"'> mais avant de faire le grand saut des images il faut savoir qu'il y a trois types d'images le .Jpg donc les photos dans l'ensemble, le .Png qui lui sert à gérer , des logos et des dessins, effin il nous reste le gif qui est pour toutes les illustrations animé

2.6 id et class


id et class ne sont pas dans des balises mais a l'interieur d'autre balise, oui s'est compliquer dit comme cela mais regarder l'exemple ci-dessoue et vous comprendrez mieux, ici il est question des class. Pour faire simple les class sont faite pour donner un nom a plusieur balise comme sur l'image. alors que les id sont fait pour que une seule balise est ce nom.

3. Commencer a codé du css


3.1 Comment mettre des paragraphe ou autre a différant endroi de la page


Pour commencer il faut que vous marquiez la même chose que sur l'image en-dessous de la balise <"meta chartes="UTF-8"'>. Vous devrez enregistrer votre page en .Css, ensuite sur cette page de Css vous marquerez le nom de votre balise comme pour les liens ou p pour les paragraphes, etc. Puis vous appuierez sur les touches suivantes: cdm, maj et 5, il y aura des majuscules comme ceci {} à l'intérieur vous mettrez text align: centre (pour mettre au centre), right(pour mettre à droite) et left(pour mettre à gauche).

3.2 Comment mettre une couleur de fond à votre site


pour mettre une couleur de fond à votre site vous devez mettre dans votre page Css header puis un espace et des{} à l'intérieur vous devrez mettre background-color: black; . vous pouvez essayer votre couleur de fond de votre site sera de la couleur de vote choix. Mais attention les couleurs doivent être en anglais sinon la page de code ne comprendra pas et vous mettra du blanc. Pour finir, je vais vous montrer comment mettre une image en fond pour cela il faut mettre à la place du background-color: Black; background: URL('votre nom de dossier'). Maintenant pour retenir tout cela il vous faudra un petit exercice : pour cet exercice vous devrez premièrement mettre votre fond en noir puis mettre vos paragraphes en blanc. Cet exercice est corrigé mais faite le et vous serez devenue quelqu'un qui connaît bien les bases.
